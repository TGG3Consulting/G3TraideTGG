### 2.3 ФОРМУЛА ИТОГОВОГО SCORE

```
total_score = max(0, min(100, СУММА_ПОЗИТИВНЫХ + СУММА_НЕГАТИВНЫХ))

СУММА_ПОЗИТИВНЫХ = (
    oi_growth + oi_stability +
    funding_cheap + funding_gradient +
    crowd_bearish + crowd_bullish +
    coordinated_buying + volume_accumulation +
    cross_oi_migration + cross_price_lead +        # всегда 0 для исторических данных
    spot_bid_pressure + spot_ask_weakness + spot_imbalance_score +
    futures_bid_pressure + futures_ask_weakness + futures_imbalance_score +
    orderbook_divergence
)

СУММА_НЕГАТИВНЫХ = (
    wash_trading_penalty +      # -25 или 0
    extreme_funding_penalty +   # -15, -10 или 0
    orderbook_against_penalty   # -8, -16 или 0
)
```

**Теоретический максимум:** ~115 баллов (ограничен до 100)
**Теоретический минимум:** -48 баллов (ограничен до 0)

## 3. РАСЧЁТ ВЕРОЯТНОСТИ (PROBABILITY)

### 3.1 Базовая вероятность от Score

```
┌──────────────────┬────────────────┐
│ Accumulation     │ Базовая        │
│ Score            │ вероятность    │
├──────────────────┼────────────────┤
│ < 50             │ 45%            │
│ 50 - 64          │ 55%            │
│ 65 - 74          │ 62%            │
│ 75 - 84          │ 70%            │
│ 85 - 94          │ 78%            │
│ >= 95            │ 85%            │
└──────────────────┴────────────────┘
```

### 3.2 Корректировки вероятности

**ДЛЯ LONG СИГНАЛОВ:**
```
+5%  если  oi_change_5m > 0           (OI растёт прямо сейчас)
+5%  если  funding_rate_pct < 0       (отрицательный фандинг)
+5%  если  short_account_pct > 55%    (толпа в шортах)
```

**ДЛЯ SHORT СИГНАЛОВ:**
```
+3%  если  oi_change_5m < -2%         (OI падает резко)
+5%  если  funding_rate_pct >= 0.05%  (экстремальный фандинг)
+5%  если  long_account_pct > 60%     (толпа в лонгах)
```

**БОНУСЫ ОТ ORDERBOOK:**
```
+5%  если  spot_imbalance >= 0.4      (сильный Bid на споте)
+3%  если  spot_imbalance >= 0.2      (умеренный Bid)
+5%  если  futures_imbalance >= 0.4   (сильный Bid на фьючерсах)
+3%  если  futures_imbalance >= 0.2   (умеренный Bid)
+3%  если  spot_imb > 0.1 AND fut_imb > 0.1  (оба согласны)
-5%  если  spot_imb и fut_imb разнонаправлены (дивергенция)
+5%  если  orderbook_total >= 20      (сильный общий стакан)
+3%  если  orderbook_total >= 10      (умеренный стакан)
```

### 3.3 Максимальная вероятность по Confidence

```
┌───────────────────┬─────────────────────┬────────────────┐
│ Confidence Level  │ Score для уровня    │ Max Probability│
├───────────────────┼─────────────────────┼────────────────┤
│ LOW               │ score < 65          │ 55%            │
│ MEDIUM            │ 65 <= score < 80    │ 70%            │
│ HIGH              │ 80 <= score < 90    │ 85%            │
│ VERY_HIGH         │ score >= 90         │ 95%            │
└───────────────────┴─────────────────────┴────────────────┘
```

### 3.4 Итоговая формула

```
probability = min(max_prob_by_confidence, max(0, base_prob + adjustments))
```

## 4. ОПРЕДЕЛЕНИЕ НАПРАВЛЕНИЯ (LONG/SHORT)

Система подсчитывает "голоса" за LONG и SHORT, затем решает.

### 4.1 Сигналы за SHORT

```
┌───────────────────────────────────────────────────────────┬────────┐
│ Условие                                                   │ Голоса │
├───────────────────────────────────────────────────────────┼────────┤
│ funding_rate_pct >= 0.03%                                 │ +1     │
├───────────────────────────────────────────────────────────┼────────┤
│ funding >= 0.03% AND oi_change_1h < -5%                   │ +1     │
│ (высокий фандинг + OI падает = лонги закрываются)         │        │
```

## 6. РАСЧЁТ STOP LOSS

### 6.1 Оценка волатильности

```python
# Приоритет 1: Daily ATR для сигналов >= 4h
if valid_hours >= 4 AND atr_daily_is_real:
    volatility = atr_daily_pct

# Приоритет 2: Hourly ATR из spot_state
elif atr_is_real:
    volatility = atr_1h_pct

# Приоритет 3: ATR из futures_state
elif futures_atr_is_real:
    volatility = futures_atr_1h_pct

# Fallback: историческая волатильность
else:
    prices = last 60 close prices
    volatility = (max - min) / avg * 100

# Корректировка на изменение OI
if abs(oi_change_1h) > 10%:
    volatility *= 1.2

# Ограничение
volatility = clamp(volatility, 0.5, 20.0)
```

### 6.2 Формула Stop Loss

```
atr_sl = volatility * 1.5
sl_pct = clamp(atr_sl, 2.0, 8.0)   # Минимум 2%, максимум 8%

ДЛЯ LONG:
    stop_loss = entry_price * (1 - sl_pct / 100)

ДЛЯ SHORT:
    stop_loss = entry_price * (1 + sl_pct / 100)
```

### 6.3 Расчёт ATR (Average True Range)

```
ATR рассчитывается по 14 последним свечам:

Для каждой свечи i:
    true_range[i] = max(
        high[i] - low[i],
        abs(high[i] - close[i-1]),
        abs(low[i] - close[i-1])
    )

ATR = mean(true_range)
ATR_pct = ATR / close * 100
```

## 7. РАСЧЁТ TAKE PROFIT

### 7.1 Множитель от Score

```python
score_clamped = clamp(accumulation_score, 65, 100)
tp_multiplier = 1.0 + (score_clamped - 65) / 35 * 0.5

# При score = 65: tp_multiplier = 1.0
# При score = 100: tp_multiplier = 1.5
```

### 7.2 Уровни Take Profit

```
risk = abs(entry_price - stop_loss)

┌───────┬─────────────────────────────────────────┬─────────┐
│ Уровень│ Формула                                │ Portion │
├───────┼─────────────────────────────────────────┼─────────┤
│ TP1   │ entry ± risk * 1.2 * tp_multiplier     │ 35%     │
│ TP2   │ entry ± risk * 2.5 * tp_multiplier     │ 40%     │
│ TP3   │ entry ± risk * 4.0 * tp_multiplier     │ 25%     │
└───────┴─────────────────────────────────────────┴─────────┘

Для LONG: entry + distance
Для SHORT: entry - distance
```

### 7.3 Примеры

```
Пример при Score = 70, SL = 3%, Entry = $100:

tp_multiplier = 1.0 + (70-65)/35*0.5 = 1.071
risk = $3 (3% от $100)

TP1 = $100 + $3 * 1.2 * 1.071 = $100 + $3.86 = $103.86 (3.86%)
TP2 = $100 + $3 * 2.5 * 1.071 = $100 + $8.03 = $108.03 (8.03%)
TP3 = $100 + $3 * 4.0 * 1.071 = $100 + $12.85 = $112.85 (12.85%)
```

## 8. РАСЧЁТ RISK/REWARD

### 8.1 Формула

```
risk = abs(entry_price - stop_loss)

weighted_reward = SUM(
    abs(tp_price - entry_price) * portion / 100
    for each TP
)

risk_reward = weighted_reward / risk
```

### 8.2 Пример

```
Entry = $100, SL = $97 (risk = $3)
TP1 = $103.86 (portion 35%)
TP2 = $108.03 (portion 40%)
TP3 = $112.85 (portion 25%)

weighted_reward =
    $3.86 * 0.35 +
    $8.03 * 0.40 +
    $12.85 * 0.25
  = $1.35 + $3.21 + $3.21
  = $7.77

risk_reward = $7.77 / $3 = 2.59
```
